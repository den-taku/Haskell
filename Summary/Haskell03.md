# 第3章 型と型クラス

**型** : 互いに関連する値の集合
 - T -> F T型からF型へ変換する関数
 - v :: T vはT型の値という意味
   - 評価前の値にも使える
 - 全ての式に型がつく
   - **型推論** によって決定(型つけ規則)
   - したがってHaskellは **型安全**

**基本型**
 - **Bool**
   - FalseとTrue
 - **Char**
   - シングルクォート
 - **String**
   - ダブルクォート
 - **Int**
   - 固定長
 - **Integer**
   - 多倍長整数
     - 上限・下限なし
 - **Float**
   - 単精度浮動小数点数
 - **Double**
   - 倍精度浮動小数点数
